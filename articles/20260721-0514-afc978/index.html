Thermage FLX 單發耗材成本 28 鎂，你付的溢價 85% 全在買那張原廠防偽貼紙的心理安慰。

版主　渡鴉

那顆紫色探頭出廠的時候，成本結構裡最貴的其實是那條感應導線和過期的倒數晶片。Thermage FLX 四代探頭 900 發的官網進價，攤提下來單發耗材成本大約就在 28 美金上下。妳在診所刷卡付的那六、七萬塊，扣掉醫師鐘點、房租水電，剩下那 85% 的溢價，買的真的不是那股熱能，是妳掃描 QR Code 看到「正貨」兩個字時，大腦分泌的那點多巴胺。

美金 28 元換算過來不到台幣一千塊。一個探頭只能用兩小時，這就是典型的耗材金融學。原廠把晶片鎖死，時間一到探頭就作廢，這招比印表機墨水匣還狠。妳以為妳買的是膠原蛋白增生，其實妳是在幫這套精密防偽系統付廣告費。那個所謂的「原廠防偽貼紙」，在二手工廠的收購價比妳想像中還要高，因為只要有了那張紙，裡面裝的是翻新頭還是水貨頭，諮詢師都能看著妳的眼睛說它是真的。

很多女孩坐在診間，看著護理師當面拆封、掃碼，就覺得這筆錢花得值了。邏輯漏洞就在這：掃碼掃出的是「這枚標籤」是真的，不代表機器射出來的能量波形是穩定的。這就像買愛馬仕，妳掃了防塵袋上的序號說是正貨，但包包本人可能早在報關前就被掉包了。溢價買的是安慰劑效應，因為如果不賣這麼貴，妳反而不敢打，怕便宜沒好貨，這就是心理戰。

同樣的能量傳導，放在不同的市場定價差得離譜。在某些東南亞國家或特定免稅區域，Thermage FLX 的療程價格可以壓到台灣的一半，難道那裡的探頭成本比較低？不是，是那裡的消費者還沒被洗腦到願意為了一張貼紙付 85% 的保護費。原廠在不同地區的定價策略，完全是看當地女性對「防偽」這件事的集體焦慮程度。越焦慮的地方，溢價就越高。

妳有沒有想過，為什麼每次打完都要讓妳把空探頭帶回家？那顆塑膠殼除了佔位子，唯一的用途就是讓妳覺得「交易完成了」。事實上，回收探頭的產業鏈一直都在，更換晶片、重新封裝，技術門檻低到妳會怕。溢價之所以存在，是因為原廠需要維持一個高貴的形象，讓妳覺得這不僅是醫療，還是一種階級消費。

如果今天把所有品牌標籤都撕掉，讓妳盲測 Thermage FLX 和其他韓系、歐系散熱機制差不多的電波，妳的皮膚根本分不出 28 美金和 200 美金的差異。熱能就是熱能，膠原蛋白受熱收縮的物理性質不會因為妳掃了 QR Code 就變得比較高尚。妳付出的那幾萬塊價差，流向了跨國公司的行銷部門，而不是妳臉上的真皮層。

有些診所打著「保證原廠」的旗號收高價，其實是在賣一種「免於恐懼」的保險。這種保險的費率高得驚人，且不保證效果。當妳在糾結要不要加那幾千塊升級所謂的「官網認證診所」時，妳已經掉進了原廠設計好的行銷閉環。他們創造了假貨的恐懼，再把正貨的溢價包裝成安全感賣給妳。

這場金錢遊戲裡，最聰明的永遠是掌握晶片計時器的原廠。他們不需要保證妳打完年輕十歲，只要保證那張貼紙是真的就行了。妳臉上的紅腫會退，錢包裡的溢價不會回來。下次躺在美容床上，看著那道紫色閃光，試著算一下 28 美金跳一次的感覺，那種肉痛感，可能比電波本身更有提拉效果。
